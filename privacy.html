개인정보처리방침

1. 수집하는 개인정보

TypeScript 오류 분석기는 사용자의 개인정보를 직접 수집하지 않습니다. 모든 오류 분석은 사용자의 브라우저에서 로컬로 처리되며, 서버로 전송되지 않습니다.

2. 쿠키 및 추적 기술

본 웹사이트는 다음과 같은 목적으로 쿠키를 사용할 수 있습니다:

  • 웹사이트 이용 통계 분석 (Google Analytics)
  • 광고 서비스 제공 (Google AdSense 등)

사용자는 브라우저 설정을 통해 쿠키 사용을 거부할 수 있습니다.

3. 제3자 서비스

본 웹사이트는 다음과 같은 제3자 서비스를 이용합니다:

  • GitHub Pages (호스팅)
  • Google Analytics (방문 통계)
  • 광고 네트워크 (수익화)

각 서비스의 개인정보처리방침은 해당 서비스 제공자의 웹사이트에서 확인하실 수 있습니다.

4. 데이터 보안

입력하신 TypeScript 오류 메시지는 브라우저 내에서만 처리되며, 외부 서버로 전송되거나 저장되지 않습니다.

5. 문의

개인정보처리방침에 관한 문의사항이 있으시면 GitHub 저장소의 Issues를 통해 연락해 주세요.

6. 방침 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있으며, 변경 시 웹사이트를 통해 공지합니다.
